2. SDLC Overview

The Software Development Life Cycle (SDLC) is a step-by-step process used for planning, creating, testing, and deploying systems. It includes:

  • Requirement Gathering & Analysis
  • System Design
  • Implementation / Development
  • Testing
  • Deployment
  • Maintenance
